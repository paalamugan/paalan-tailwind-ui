# Icons

This is a collection of icons used in the package. You can find all the icons in [here](/docs/icons-all-icons--docs).

## Usage

- Import the **Icons** you want to use from `@paalan/tailwind-ui/icons`.

```tsx
import { AccessibilityIcon } from '@paalan/tailwind-ui/icons';

<AccessibilityIcon />;
```
