# Icons

This is a collection of icons used for the applications. They are available in both outline and solid styles. The icons are available as React components.

## Usage

- Import the **Custom icon** you want to use from `@paalan/tailwind-ui/icons`.

```tsx
import { AccessibilityIcon } from '@paalan/tailwind-ui/icons';

<AccessibilityIcon />;
```

- Import the **Outline icon** you want to use from `@paalan/tailwind-ui/icons/outline`.

```tsx
import { AcademicCapIcon } from '@paalan/tailwind-ui/icons/outline';

<AcademicCapIcon />;
```

- Import the **Solid icon** you want to use from `@paalan/tailwind-ui/icons/solid`.

```tsx
import { AcademicCapIcon } from '@paalan/tailwind-ui/icons/solid';

<AcademicCapIcon />;
```
