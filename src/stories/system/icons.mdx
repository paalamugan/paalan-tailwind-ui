# Icons

This is a collection of icons used for the package. You can find all the icons in [here](/docs/icons-allicons--docs). The icons are available as React components.

## Usage

- Import the **Icons** you want to use from `@paalan/tailwind-ui/icons`.

```tsx
import { AccessibilityIcon } from '@paalan/tailwind-ui/icons';

<AccessibilityIcon />;
```

## List Icons

- All icons are available as React components in the `@paalan/tailwind-ui/icons` package.

<IconGallery>
  {Object.entries(allIcons).map(([name, Icon]) => (
    <IconItem key={name} name={name}>
      <Icon />
    </IconItem>
  ))}
</IconGallery>
