# useUpdateEffect

`useUpdateEffect` hook works the same way as `useEffect` but it is not called when component is mounted.

```ts
import { useUpdateEffect } from '@glidwell/tailwind-ui/hooks';

function Usage() {
  useUpdateEffect(() => console.log("Won't be called when mounted"), [value]);
}
```
