# useFocusTrap

`useFocusTrap` is a React hook that traps focus within a specified element. It returns a function that returns focus to the last active element.

## Parameters

- `active` - Whether or not the focus trap is active. Defaults to `true`.

## Returns

`useFocusTrap` returns a function that returns focus to the last active element.
