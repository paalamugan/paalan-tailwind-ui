# useHover

`useHover` is a React hook that tracks whether a component is being hovered over.

## Parameters

`useHover` accepts the no parameters.

## Returns

`useHover` returns a object with the following properties:

- `hovered` - a boolean indicating whether the component is being hovered over
- `ref` - a React ref that should be attached to the component's root element
