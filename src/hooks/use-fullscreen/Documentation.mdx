## Parameters

`useFullscreen` does not take any parameters.

## Returns

`useFullscreen` returns an object with the following item:

- `toggle` - a function that toggles fullscreen mode
- `fullscreen` - a boolean that indicates whether fullscreen mode is active
- `ref` - a React ref that should be attached to the element that should be fullscreened
