# useTextSelection

`useTextSelection` is a React hook that allows you to get and set the current text selection in a DOM element. It is useful for building rich text editors.

## Parameters

`useTextSelection` does not take any parameters.

## Returns

`useTextSelection` returns the current text `Selection` object, or `null` if there is no text selection.
