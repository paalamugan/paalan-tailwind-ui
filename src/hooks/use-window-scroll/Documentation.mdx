# useWindowScroll

`useWindowScroll` returns current scroll position and a function to scroll smoothly to given position.

## Parameters

`useWindowScroll` accepts no parameters.

## Returns

`useWindowScroll` returns an tuple with following elements:

- `position` - current scroll position

  - `x` - horizontal scroll position
  - `y` - vertical scroll position

- `scrollTo` - function to scroll smoothly to given position with following parameters:
  - `x` - horizontal scroll position
  - `y` - vertical scroll position
