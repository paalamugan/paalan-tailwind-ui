# useLogger

`useLogger` logs given values to the console each time component renders. Open devtools to see state changes in console.

### Example

### Code

## Parameters

`useLogger` accepts two parameters:

- `componentName` - a string containing the name of the component to be logged
- `props` - an object containing the props to be logged

## Returns

`useLogger` returns nothing.
