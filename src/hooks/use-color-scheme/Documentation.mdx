# useColorScheme

`useColorScheme` is a React hook that returns the current color scheme of the user's operating system. It is useful for implementing dark mode in your application.

## Parameters

`useColorScheme` accepts no parameters.

## Returns

`useColorScheme` returns a string that is either `light` or `dark`.
