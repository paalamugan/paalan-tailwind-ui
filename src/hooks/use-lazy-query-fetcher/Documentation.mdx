# useLazyQueryFetcher

`useLazyQueryFetcher` is a hook that allows you to fetch data lazily. It is useful when you want to fetch data only when it is needed.

## Parameters

`useLazyQueryFetcher` accepts the single parameter `queryFetcher`.

## Returns

`useLazyQueryFetcher` returns an object with the following properties:

- `data` - The data returned by the handler.
- `error` - The error returned by the handler.
- `fetcher` - A function that will trigger the handler to be called again.
- `initialized` - A boolean indicating whether the handler has been called at least once.
- `loading` - A boolean indicating whether the handler is currently being called.
