# useEyeDropper

`useEyeDropper` is a React hook that allows you to get the color of a pixel on the screen.

## Parameters

`useEyeDropper` accepts no parameters.

## Returns

`useEyeDropper` returns an object with the following properties:

- `supported` - A boolean indicating whether the browser supports the `getPixelColor` API.
- `open` - A function that opens the eye dropper.
