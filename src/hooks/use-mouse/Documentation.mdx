## Parameters

`useMouse` accepts a single parameter, an object of options.

- `resetOnExit` - Whether to reset the mouse position to `(0, 0)` when the mouse leaves the tracked element. Defaults to `false`.

## Returns

`useMouse` returns an object with the following properties:

- `ref` - A `React.RefObject` that should be attached to the element that you want to track the mouse position and state for.
- `x` - The current x position of the mouse relative to the element.
- `y` - The current y position of the mouse relative to the element.
