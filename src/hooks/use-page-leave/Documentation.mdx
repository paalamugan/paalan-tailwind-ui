# usePageLeave

`usePageLeave` is a React hook that allows you to execute a function when the user leaves the page.

## Parameters

`usePageLeave` accepts a function as its only parameter. This function will be executed when the user leaves the page.

## Returns

`usePageLeave` returns nothing.
