## Parameters

- `eventType` - The event type to listen for.
- `handler` - The event handler to attach to the event.
- `options` - An object containing options to pass to the event listener.
  - `capture` - A boolean indicating whether the event should be captured.
  - `passive` - A boolean indicating whether the event should be passive.
  - `once` - A boolean indicating whether the event should only be fired once.
- `target` - The target element to attach the event listener to. Defaults to `window`.

## Returns

`useEventListener` returns a mutable ref object that can be attached to a React component.
