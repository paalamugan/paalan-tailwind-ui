# useWindowEvent

`useWindowEvent` adds event listener to window object on component mount and removes it on unmount.

```tsx
import { useEffect } from 'react';

import { useWindowEvent } from '@paalan/tailwind-ui/hooks';

const handler = (event) => console.log(event);

// regular way
useEffect(() => {
  window.addEventListener('keydown', handler);
  return () => window.removeEventListener('keydown', handler);
}, []);

// with use-window-event hook
useWindowEvent('keydown', handler);
```

## Parameters

`useWindowEvent` accepts three parameters:

- `type` - event type
- `handler` - event handler
- `options` - event options or `true` for `once` option
  - `once` - if `true`, the listener will be automatically removed after it triggers the first time
  - `passive` - if `true`, the listener will not call `preventDefault()` on the event

## Returns

`useWindowEvent` returns nothing.
