# useFocusReturn

`useFocusReturn` is a React hook that returns focus to the last active element when a component is unmounted or closed. Its used in Modal, Drawer and etc.

## Parameters

- `options` - An object containing the following properties:
  - `opened` - A boolean indicating whether the element is opened or not.
  - `shouldReturnFocus` - A boolean indicating whether the element should return focus or not.

## Returns

`useFocusReturn` returns a function that returns focus to the last active element.
