# useIsomorphicEffect

`useIsomorphicEffect` is a replacement for useEffect hook that works in both browser and server environments.

## Parameters

`useIsomorphicEffect` accepts the same parameters as `useEffect` and `useLayoutEffect`. See the [React documentation](https://reactjs.org/docs/hooks-reference.html#useeffect) for more information.

## Returns

`useIsomorphicEffect` returns the same value as `useEffect` and `useLayoutEffect`. See the [React documentation](https://reactjs.org/docs/hooks-reference.html#useeffect) for more information.
