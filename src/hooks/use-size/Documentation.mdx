# useSize

`useSize` is a React hook that measures the size of an element and returns it as an object with `width` and `height` properties. It also returns a `ref` that you need to assign to the element you want to measure.

## Parameters

`useSize` accepts an `ref` as its only parameter. You need to assign this `ref` to the element you want to measure.

## Returns

`useSize` returns an object with the following properties:

- `width` - the width of the element in pixels
- `height` - the height of the element in pixels
